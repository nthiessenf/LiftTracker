# LiftTrack Design System

## Reference Mockup
See `mockups/ui-redesign-mockup.jsx` for the interactive UI redesign target.

## Design Tokens

### Colors
- Background: #0a0a0a to #121212 (gradient)
- Card Background: rgba(255,255,255,0.05) to rgba(255,255,255,0.08)
- Card Border: rgba(255,255,255,0.1)
- Primary Accent: #10b981 (green)
- Primary Gradient: linear-gradient(135deg, #10b981 0%, #059669 100%)
- Text Primary: #FFFFFF
- Text Secondary: rgba(255,255,255,0.5)
- Text Muted: rgba(255,255,255,0.4)

### Typography
- Title: 32-38px, font-weight 700, letter-spacing -1px
- Heading: 18-20px, font-weight 600
- Body: 15-17px, font-weight 400
- Caption: 12-14px, font-weight 400-500

### Spacing
- Screen padding: 24px
- Card padding: 20px
- Card border-radius: 20px
- Button border-radius: 12px
- Section gap: 24-32px

### Effects
- Card shadow: 0 8px 32px rgba(0,0,0,0.3)
- Card inner highlight: inset 0 1px 0 rgba(255,255,255,0.1)
- Button glow: 0 4px 16px rgba(16,185,129,0.4)
- Backdrop blur: blur(20px)

This file documents the design system for reference when building components.
